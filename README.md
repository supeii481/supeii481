### <img width="40" alt="image" src="https://tibamef2e.com/cjd102/g2/api/img/site/Recimo_avatar.svg" /> [小組專題] Recimo-食譜筆記網站
**[🔗 Demo](https://tibamef2e.com/cjd102/g2/) | [🎬 影片演示](https://www.youtube.com/watch?v=Ti9WaaihIWA) | [📜 專案簡報](https://drive.google.com/file/d/1JSVifu9X9QTcAMhD43dmvcUeuZNUT-qP/view) | [📄 系統文件](https://drive.google.com/file/d/1KFBa0OSVV8k4k5qAoByQz0BYtJy0y0Xz/view)**


- **開發動機**：從觀察出發的技術實踐
  - 本專案發想於我對親友記錄食譜困境的觀察，傳統紙本易丟失、Notion 等工具門檻過高，於是我將行政實務中的「優化流程」思維導入專案，提議打造一個「直覺且強大」的筆記平台，成功獲得團隊採納並立案。
    
- **專題簡介**：
  - 我們致力於打造一個結合「線上食譜圖書館」與「高自由度數位筆記」的創新平台。
  - **解決痛點**：克服紙本筆記零散與數位複製貼上缺乏彈性的問題。
  - **目標願景**：讓每一次烹飪都不只是體力活，而是一次「高效、智慧且可追蹤」的美味創作。

- **專案貢獻與核心技術**：
  - **UI/UX 設計**：
    - 使用 Figma 獨立規劃前後台入口、關於 Recimo、會員權益、前台登入/註冊燈箱、會員中心、通知及後台登入之 Wireframe、Mockup。
  - **前後台開發**：
    - 主導前後台入口頁、會員系統、商城模組（商品卡片、詳細頁、購物車）、關於Recimo頁面及後台登入的開發。
    - **商城模組**：實現商品卡片渲染、商品詳細頁邏輯、購物車增刪改查功能。
    - **完整會員系統**：開發包含登入註冊、忘記密碼流程、個人頭貼全站連動更新及會員中心。
  - **核心技術運用**：
    - **前端框架**：Vue 3 組合式 API、SCSS 模組化樣式。
    - **狀態管理**：使用 **Pinia** 處理跨組件資料同步（如會員資訊、購物車狀態）。
    - **路由控制**：使用 **Vue Router** 實作動態路由與**路由守衛 (Navigation Guards)** 確保權限安全。
    - **後端整合**：使用 **PHP** 撰寫 API 與 **MySQL** 資料庫進行前後端資料對接。
  - **第三方功能整合**：
    - **社群登入**：串接 **Google / Facebook / Line** 第三方登入 API。

### <img width="40" alt="image" src="https://raw.githubusercontent.com/supeii481/Zusic_website/6423656f65ef2be00c879b1fe50d328feb356bdb/img/Zusic-logo-small.svg" /> [個人專題] Zusic租樂趣-快速租借樂器網站
**[🔗 Demo](https://supeii481.github.io/Zusic_website/index.html) | [📜專案簡報](https://drive.google.com/file/d/1e3z7I6nADvUh1NM-oMA0vWk9NR-C0FGt/view?usp=drive_link)**

- **開發亮點**
  - **成長軌跡**：此網站是我前端進修初期（學習未滿兩個月）的作品。
  - **跨越挑戰**：在極短時間內從設計及程式新手，高強度同步吸收新知，獨立完成從 UI 設計到 RWD 網站開發的首項個人專案。
- **網站簡介**
  - 旨在降低學習樂器的入門門檻。針對「想練習但無樂器」或「樂器不便攜帶」的痛點，打造一個能方便、快速借到樂器的平台，讓使用者即時享受音樂帶來的樂趣。

- **品牌設計理念 (Logo Design)**：我將「自由、動感、專業」三種元素融入品牌識別中
  - **自由與溫度**：大筆觸漸層字母 **Z**，象徵租借規定的高度自由；由藍轉橘的漸層色調，強調年輕、創意與有溫度的服務形象。
  - **音樂與即時性**：右側結合**音波/波形**設計，暗示平台的即時性（聲波動感 = 快速租借），直觀傳遞音樂核心。
  - **產業辨識度**：融入**鋼琴黑鍵**形狀，強化樂器租借平台的專業第一印象。
