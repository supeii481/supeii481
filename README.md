### <img width="40" alt="image" src="https://tibamef2e.com/cjd102/g2/api/img/site/Recimo_avatar.svg" /> [小組專題] Recimo 食譜筆記網站
#### DEMO：https://tibamef2e.com/cjd102/g2/
#### DEMO 影片：https://www.youtube.com/watch?v=Ti9WaaihIWA

- **開發動機**：源於自身親友的真實需求，希望能建立一個直覺好上手且具備筆記功能的食譜筆記平台。
- **專題簡介**：我們深知紙本筆記容易丟失、數位貼上又缺乏彈性，尤其當想在食譜上自由編輯筆記、新增圖片註記時，會極為不便。因此我們希望打造一個如notion結合線上食譜圖書館般自由的數位研究室，讓使用者不僅能輕鬆紀錄，也能精準掌握配方差異，讓每一次烹飪都是一次高效、智慧且可追蹤的美味創作。 
 - **技術重點**：
●Web/APP UI/UX設計 [使用者介面設計、網站流程動線規劃、wireframe、mockup、prototype
●網頁設計 [HTML、CSS、RWD行動網頁(含Bootstrap)] 
●前端網頁技術 [HTML5、CSS3動態特效、Javascript、jQuery、AJAX、社群API、SASS、Vue.js  
●網站前後端功能開發與資料庫管理 [系統分析、PHP、MySQL、Git版本控制] 


- **技術亮點**：
  - 負責從規劃設計、UI 到前後台完整開發流程。
  - 使用 Vue 3 + Pinia 進行狀態管理。
  - 整合 Google/Facebook 第三方登入功能。
  - 實作高度結構化的 CRUD 管理介面，展現行政邏輯轉化成果。
這是一個完整的團隊協作專案，我參與了以下部分的開發：
* **Recimo Backend**: 負責後端架構與資料庫設計 (Node.js/PHP/MySQL)。
* **Recimo Admin**: 開發管理後台，確保資料管理流程順暢。
* **Recimo Main Website**: 參與前端介面開發與互動邏輯實作。
